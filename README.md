# chat_app
## Microsoft Engage 2021

A fully functional Flutter Application with Chatting(group/single), Video Calling .

## Getting Started
- `git clone https://github.com/gauriwanare/FlutterApp.git`
- `cd FlutterApp`
- `flutter packages get`
- `flutter run`
